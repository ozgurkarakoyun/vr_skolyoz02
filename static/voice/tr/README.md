# Türkçe doğal ses dosyaları

Program önce bu klasördeki MP3 dosyalarını arar. Dosya yoksa tarayıcının Türkçe Web Speech API sesini kullanır.

İsteğe bağlı dosya adları:

- session_start.mp3 — "Seans başladı. Hazırsanız başlayalım."
- excellent.mp3 — "Harika. Mükemmel pozisyon."
- continue.mp3 — "Harikasınız, böyle devam edin."
- posture_better.mp3 — "Çok güzel, duruşunuz belirgin şekilde iyileşiyor."
- very_good.mp3 — "Çok iyi, devam edin."
- breathe_in.mp3 — "Nefes alın."
- hold.mp3 — "Tutun."
- breathe_out.mp3 — "Nefes verin."
- warning.mp3 — "Dikkat. Pozisyonu kontrol edin."

Daha doğal ses istenirse bu cümleler gerçek insan sesiyle veya kaliteli Türkçe TTS ile kaydedilip aynı isimlerle buraya eklenebilir.
